販促コンペ2023
KOKUYO SET MAKER

- 2023.04.20~2023.06.06 -

販促コンペ2023のうちの課題のひとつ、株式会社コクヨの「中高生がまるっとコクヨの文具達を揃えたくなっちゃうアイデア」の企画。今回は企画案をWEBサービスをとして制作。中高生の利用シーンやサービスに至るまでの導線などを考えて制作しました。

中高生が文具に興味を持つキッカケとは何かを重点的に考えました。今回の企画では、自分の文具セットをサービス内で作成してもらい、そのままセットごと購入できるというサービスにしました。また、SNSでの拡散要素も取り入れられるよう、購入後画面に人によって違うグラフィックが現れるという機能を追加しました。
